Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Di mana :
L = Luas
a = Panjang Alas
t = Tinggi

Hitung Luas Segitiga

Masukan Alas

Masukan Tinggi

Luas Segitiga Adalah

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Di mana :
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C

Hitung Keliling Segitiga

Masukan Sisi Pertama

Masukan Sisi Kedua

Masukan Sisi Ketiga

Keliling Segitiga Adalah

Luas Jajar Genjang

Rumus Luas Jajargenjang yaitu :

L = a x t

Di mana :
L = Luas
a = Alas
t = Tinggi

Hitung Luas Jajar Genjang

Masukan Alas

Masukan Tinggi

Luas Jajar Genjang Adalah

Keliling Jajar Genjang

Rumus Keliling Jajar Genjang yaitu :

K = 2 (a + m)

Di mana :
K = Keliling
a = Alas
m = Sisi Miring

Hitung Keliling Jajar Genjang

Masukan Alas

Masukan Sisi Miring

Keliling Jajargenjang Adalah
